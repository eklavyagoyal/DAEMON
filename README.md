## 🧠 What is DAEMON?

DAEMON (pronounced "demon" — Unix background process) is a **managed AI operator**. Think of it as a 24/7 employee that:

- ✅ Lives on a VPS (Hetzner, €4-8/mo)
- ✅ You control via **Telegram** from your phone
- ✅ Has **persistent memory** across sessions
- ✅ Browses the web, runs code, edits files
- ✅ Runs **scheduled cron jobs** (daily reports, market scans)
- ✅ Switches between **232 expert personas** instantly
- ✅ Costs ~€15-25/mo in API fees (DeepSeek Flash)

It's not a chatbot. It's an **operator**.

## ❓ FAQ

**What is DAEMON exactly?**
A managed Hermes Agent running 24/7 on a VPS. You control it through Telegram. It thinks, uses tools, remembers context, and works autonomously.

**How is this different from ChatGPT?**
DAEMON is autonomous and persistent. It runs code, edits files, browses the web, manages cron jobs, and works while you sleep. ChatGPT is a chat. DAEMON is an operator.

**Who pays for API costs?**
You bring your own API key. Typical DeepSeek Flash usage: €15-25/mo. We never touch your key.

**How fast can I get it?**
Lite: 24h. DAEMON: 48h. Pro: custom. We provision, configure, connect, and hand you the keys.

**Can I cancel?**
Yes. Monthly, no contract. You keep the VPS and all data.
