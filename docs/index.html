Frequently Asked Questions

What is DAEMON exactly?
DAEMON is a managed Hermes Agent running on a dedicated VPS. You control it through Telegram — it thinks, uses tools, browses the web, runs code, and reports back. Think of it as a 24/7 AI operator that works for you.
How is this different from ChatGPT or Claude?
DAEMON is autonomous and persistent. It has memory across sessions, can run code, edit files, browse the web, manage cron jobs, and work while you're offline. ChatGPT is a chat. DAEMON is an operator.
What about API costs? Who pays for the LLM?
You bring your own API key (DeepSeek, Anthropic, or any provider). Typical DeepSeek Flash usage for a power user runs €15-25/month. We never touch your key — zero billing risk for us, full control for you.
How fast can I get DAEMON running?
Lite: 24 hours. Standard DAEMON: 48 hours. Pro: custom timeline. We provision the VPS, configure Hermes, connect Telegram, set up your personality and tools — then hand you the keys.
Can I switch between the 232 agent personas?
Yes. Every DAEMON ships with all 232 expert personas from The Agency (116k★ GitHub). One Telegram message switches DAEMON from "Growth Hacker" to "Security Auditor" to "Frontend Developer". Each persona has full context and specialized workflows.
What platforms does DAEMON work on?
Telegram is primary (you text it from your phone). Also supports Discord, Slack, WhatsApp, Signal, Matrix, Email, and CLI. All from the same Hermes instance.
Can I cancel anytime?
Yes. Monthly hosting is paid per month — no contract. If you cancel, we give you the VPS credentials and all your data. You keep everything.
